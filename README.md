# Prova 3 — Urna Eletrônica
 
Projeto desenvolvido na disciplina de **Programação Orientada a Objetos (PROO)**.
 
Sistema simples de urna eletrônica em Java, que carrega candidatos a partir de um
dataset, registra votos para os cargos de Presidente, Governador, Senador, Deputado
Federal e Deputado Estadual, e apura os eleitos por cargo conforme o número de vagas.
